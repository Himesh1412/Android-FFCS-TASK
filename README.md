# Android-FFCS-TASK
Color Trap is a simple mobile game built using Flutter. Players must quickly decide whether the meaning of a displayed color word matches the actual color shown on the screen. The game includes a start screen, gameplay screen with timer and score, and a result screen. It tests reaction speed and attention.
